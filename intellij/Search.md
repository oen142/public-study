## 검색 텍스트

- 현재 파일에서 검색
    - 윈도우,리눅스 : `Ctrl + F`
    - 맥 : `Command + F`
- 현재 파일에서 교체
    - 윈도우,리눅스 : `Ctrl + R`
    - 맥 : `Command + R`
- 전체에서 검색
    - 윈도우,리눅스 : `Ctrl + Shift +  F`
    - 맥 : `Command + Shift + F`
- 전체에서 교체
    - 윈도우,리눅스 : `Ctrl + Shift +  R`
    - 맥 : `Command + Shift + R`
- 정큐포현식으로 검색, 교체
    - 윈도우,리눅스 : `Regex Check`
    - 맥 : `Regex Check`
