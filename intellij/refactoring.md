## 리팩토링 Extract

- 변수 추출하기
    - 윈도우, 리눅스 : `Ctrl + Alt + V`
    - 맥 : `Command + Option + V`
- 파라미터 추출하기
    - 윈도우, 리눅스 : `Ctrl + Alt + P`
    - 맥 : `Command + Option + P`
- 메소드 추출하기
    - 윈도우, 리눅스 : `Ctrl + Alt + M`
    - 맥 : `Command + Option + M`
- 이너클래스 추출하기
    - 윈도우, 리눅스 : `F6`
    - 맥 : `F6`
    
    
## 이외의 리팩토리

- 이름 일괄 변경하기
    - 윈도우, 리눅스 : `Shift + F6`
    - 맥 : `Shift + F6`
- 타입 일괄 변경하기 
    - 윈도우, 리눅스 : `Shift + Ctrl + F6`
    - 맥 : `Shift + Command +F6`
- import 정리하기 
    - 윈도우, 리눅스 : `Ctrl + Option + O`
    - 맥 : `Ctrl + Alt + O`
- 코드 자동 정렬하기 
    - 윈도우, 리눅스 : `Ctrl + Alt + L`
    - 맥 : `Command + Option + L`
