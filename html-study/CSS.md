### CSS

## 선택자의 역할

- 선택자는 HTML에 스타일(CSS)을 적용하기 위해 HTML의 특정한 요소를 선택하는 사인이다.

```css
h1{
    color:red;
}
p{
    color:blue;
}
```

## 속성(Properties) 과 값(Value)
- 속성과 값은 글자색은 무엇 너비는 얼마 여백은 얼마와 같은 스타일을 지정할때 사용한다.

- HTML에 포함하기

- HTML 외부에서 불러오기
