### HTML 태그

- 문서의 시작부분과 끝부분을 지칭

- lang 언어 설정 korea는 ko english는 en  
    - ISO 639-1 코드 목록
    
### !DOCKTYPE

- html5 사용하기 위해서 사용
    
### HEAD

- 문서의 정보 입력
    - html 문서의 제목
        - title tag
    - 기타 정보
        - meta tag
            - charset 문자가 인코딩 되는 방식
            
    - 스타일 직접 입력
    - 스타일 외부에서 가져와서 연결
    
### BODY

- 문서의 구조 입력
