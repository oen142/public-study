        - 세션 유지시 사용됨
        - 고정 비활성화
        - 로드밸런서 생성 쿠키 고정 활성화
        - 애플리케이션 생성쿠키 고정 활성화


- NLB(Least connection)  최소 연결 방식
    - 상태검사는  http를 하는게 좋다. 하지만 부하가 갈 수 있다.
    
- ALB(Round Robin) 순차적 연결 방식
    - L7 S/W = Content Switch
